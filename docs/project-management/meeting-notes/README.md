# Meeting Notes

Create one file per project meeting or planning session.

Suggested naming:

```text
2026-05-20-planning.md
```

Template:

```markdown
# YYYY-MM-DD Meeting Title

## Attendees

- Name

## Agenda

- Topic 1

## Notes

- Note 1

## Decisions

- Decision 1

## Action Items

| Item | Owner | Due | Status |
| --- | --- | --- | --- |
| Action item | Owner | TBD | Open |
```
